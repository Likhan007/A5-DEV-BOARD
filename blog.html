Question-1: What are the different ways to select an element in the DOM?

There are a few ways to select element in the DOM. getElementsByTagName, getElementByClass, getElementById, getElementByQuerrySelectorAll and many more.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML modifies the html contents inside the selected element.
innerText just sets the text content
and finally the textContent is used to return the text content of an element

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling means an event starts at the target element and moves up to its parents.

Question-5: How do you create, add, and remove elements using JavaScript?

document.createElement() is used to create element.
appendChild() can be used to append child
removeChild() or just remove() can be used to remove a child
